Para colocar um código no console na página do Instagram, siga estes passos:

1. Copie o link e cole no seu navegador.

2. Apos abrir o link copie o código do link.

3. Abra o Instagram no navegador.

4. Pressione F12 ou clique com o botão direito e selecione "Inspecionar".

5. Vá até a aba "Console".

6. Cole o código e pressione Enter.
